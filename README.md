# javascript-modules


A collection of Vanilla ES6 modules I tend to reuse.


### Requirements

- Node.js >= 10


### Getting Started

```
git clone https://github.com/pboi20/javascript-modules
cd javascript-modules
npm install
```

### Available Scripts

#### `npm run dev`

Starts the bundler in watch mode.

#### `npm run build`

Creates a production bundle at `dist/main.js`.
